# GET-ME-A-CHAI
This is a membership platform project that allows creators, such as artists, musicians, and podcasters, to earn a steady income from their fans by offering exclusive content and benefits.
A full-stack Patreon-style membership and creator-support platform built using Next.js, Node.js/Express, and MongoDB.
Users can create accounts, support creators, subscribe to membership tiers, and access exclusive content.

Technologies Used:
Frontend: Next.js, TailwindCSS, React Hooks, Fetch APi
Backend: Node.js, Express, MongoDB Atlas, Mongoose, JWT Authentication
